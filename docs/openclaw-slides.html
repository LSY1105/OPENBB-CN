什么是 OpenClaw？

What is OpenClaw
OpenClaw 是一个生产级的多 Agent AI 协作平台。一个实例支持多个 AI Agent 协同工作，每个 Agent 拥有独立的人设、技能与工作区。通过灵活的工具链集成和持久化知识库，实现跨渠道、跨任务、跨会话的智能协作。
🤖

多 Agent 系统

多个 AI Agent 各司其职、协同作战，通过 binding 配置区分不同 Agent 的能力边界与专业领域。

⚡

智能任务流

SQLite 持久化的任务账本，48 小时超长待机，统一调度 ACP 任务、Cron 定时任务与子 Agent。

🧠

持久知识库

Karpathy 模式的四阶段循环：Ingest → Compile → Query → Lint，知识编译一次，持续复利。

核心功能详解

Core Features
⚡

Task Flow 任务流

SQLite 持久化 · 48h 超长待机
  • SQLite 持久化状态 + 修订版追踪
  • /tasks 命令实时查看所有后台任务
  • ACP 任务 + Cron + 子 Agent 统一管理
  • Sticky Cancel Intent · 阻塞原因可见
  • api.runtime.taskFlow 插件开发接口
🧠

知识库 Karpathy 模式

Ingest → Compile → Query → Lint
  • Wiki 双向链接，知识编译一次持续复用
  • PDF 读取（pdfplumber）+ Markdown 编写
  • PDF 导出（reportlab）+ PPT 生成
  • 健康检查脚本：死链接 + 孤立文章
  • 可配合 Obsidian / Dataview 使用
🔗

多渠道接入

全平台覆盖 · 语音唤醒
  • 飞书 / Telegram / Discord / Signal / Matrix
  • LINE · QQ Bot · Android · macOS
  • Voice Wake 语音唤醒（macOS）
  • Google Assistant App Actions（Android）
  • WhatsApp emoji 反应 · Telegram 审批流
🔒

安全加固

企业级安全 · 危险代码扫描
  • 插件危险代码扫描（静态分析）
  • 节点配对 ≠ 命令权限（防提权攻击）
  • Shell 环境变量严格过滤
  • BCE Auth v1 时间安全比较
  • SSRF 防护 + 传输层 TLS 集中化
